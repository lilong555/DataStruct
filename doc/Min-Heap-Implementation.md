# 最小堆实现（项目七）

项目七需要反复取出当前最小的两个长度，因此适合使用最小堆（小根堆）。

## 数据结构

使用数组实现二叉堆（完全二叉树）：

- 使用 1 下标数组更便于计算父子关系
- 父节点：`i / 2`
- 左孩子：`2 * i`
- 右孩子：`2 * i + 1`

## 基本操作

### 1）上滤（SiftUp）

用于插入后恢复堆性质：

- 新元素先放到堆尾
- 若其值小于父节点，则与父节点交换
- 重复直到到达根或父节点更小

### 2）下滤（SiftDown）

用于删除堆顶后恢复堆性质：

- 取出堆顶后，将堆尾元素放到堆顶
- 在左右孩子中选择更小者与当前节点比较
- 若孩子更小则交换并继续向下

### 3）入堆/出堆

- 入堆：堆尾插入 + 上滤
- 出堆：取堆顶、堆尾补到堆顶 + 下滤
