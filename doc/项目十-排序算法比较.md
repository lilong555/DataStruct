# 项目十：8 种排序算法比较

本项目实现 8 种经典排序算法，并对它们在随机数据上的表现做对比统计：

- 运行时间（毫秒）
- 比较次数（仅对“比较类排序”有意义）
- 交换/移动次数（用于粗略衡量数据移动开销）

## 8 种排序算法

1. 冒泡排序（Bubble Sort）
2. 选择排序（Selection Sort）
3. 直接插入排序（Insertion Sort）
4. 希尔排序（Shell Sort）
5. 快速排序（Quick Sort）
6. 堆排序（Heap Sort）
7. 归并排序（Merge Sort）
8. 基数排序（Radix Sort）

## 程序交互

程序为菜单交互方式：

1. 选择算法编号（1-8）
2. 输入随机数规模 `N`
3. 输出该算法统计信息
4. 可选择是否对所有算法进行性能比较（固定规模 100/1000/10000/100000 + 自定义规模）

## 数据生成

程序内部生成 `N` 个随机整数（范围约在 `[0, 1000000)`），用于排序测试。

## 统计口径说明（简要）

- **比较次数**：发生“大小比较”的计数（如 `arr[i] > arr[j]`）
- **交换/移动次数**：对元素交换或移动的计数（不同算法口径略有差异，但可用于相对比较）
- **时间**：使用 `clock()` 统计 CPU 时间（换算为毫秒）

## 头文件限制说明
