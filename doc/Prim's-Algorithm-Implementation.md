# Prim 算法实现（项目八）

Prim 算法用于在连通无向带权图中构造最小生成树（MST）。

## 基本思想

从某个起点出发，逐步扩展“已在生成树中的顶点集合”：

1. 初始集合只包含起点
2. 每次在所有“从集合内到集合外”的边中，选择权值最小的一条
3. 将该边连接的外部顶点加入集合
4. 重复直到包含全部顶点（得到 `n-1` 条边）

## 常用数组（邻接矩阵版本）

设 `V` 为顶点数：

- `visited[i]`：顶点 `i` 是否已经加入 MST
- `lowcost[i]`：当前 MST 集合到顶点 `i` 的最小边权
- `closest[i]`：使 `lowcost[i]` 取到最小值时，对应的 MST 内顶点

每次选择 `lowcost` 最小且未访问的顶点加入 MST，并更新其他未访问顶点的 `lowcost/closest`。

## 复杂度

邻接矩阵实现通常为：

- 时间复杂度：O(V²)
- 空间复杂度：O(V²)（邻接矩阵）

## 非连通图处理

如果某一步找不到可加入的新顶点（即 `lowcost` 全为无穷大），说明图不连通，无法构造覆盖全部顶点的 MST，需要给出提示并终止构造。
