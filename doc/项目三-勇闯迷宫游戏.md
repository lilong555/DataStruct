# 项目三：勇闯迷宫游戏

本项目在二维迷宫中寻找从入口到出口的一条可行路径，采用 DFS + 回溯思想实现。

## 输入

程序按提示读入：

1. 迷宫行数、列数
2. 迷宫矩阵：`0` 表示通路，`1` 表示墙
3. 起点坐标（行、列）
4. 终点坐标（行、列）

坐标以 `0` 为起始（即第 0 行、第 0 列）。

## 输出

- 若存在路径：输出带路径标记的迷宫地图，并输出路径坐标序列
- 若不存在路径：提示无法从起点到达终点

## 核心思路

- 从起点出发，按固定方向顺序尝试移动（上下左右之一）
- 每到达一个新位置：
  - 标记为已访问（避免循环）
  - 递归继续探索
  - 若走不通则回溯，撤销当前选择
- 到达终点即结束搜索

## 编译与运行

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic p3/maze_game.cpp -o output/p3
./output/p3
```

也可以使用测试文件重定向输入：

```bash
./output/p3 < p3/test_input.txt
```
