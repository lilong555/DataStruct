# 算法参考

本仓库围绕课程设计题目实现了多种经典算法，主要包括：DFS 回溯、贪心最优合并、Prim 最小生成树、以及多种排序算法。

## 1. 哈希（Hashing）

用途：将键映射到桶索引，实现快速查找。  
典型项目：P1（考试报名系统）。

要点：

- 合理的哈希函数与桶数选择
- 冲突处理（拉链法）

## 2. 深度优先搜索（DFS）与回溯

用途：在状态空间中搜索可行解；走不通则回退尝试其他选择。  
典型项目：P3（迷宫寻路）。

要点：

- `visited` 标记避免循环
- 路径记录与状态恢复

## 3. 贪心：最优合并（哈夫曼式合并）

用途：每次合并当前最小的两个元素以获得全局最优总代价。  
典型项目：P7（修理牧场最小花费）。

要点：

- 小根堆维护最小值
- 总复杂度 O(N log N)

## 4. Prim 最小生成树

用途：在连通无向带权图中构造 MST，使总边权最小。  
典型项目：P8（电网建设造价）。

要点：

- 邻接矩阵版本常见复杂度 O(V²)
- 维护 `lowcost/closest/visited`

## 5. 排序算法

典型项目：P10（排序算法比较）。

包含：

- 比较类排序：冒泡、选择、插入、希尔、快速、堆、归并
- 非比较类排序：基数排序

对比维度：

- 时间复杂度（平均/最坏）
- 空间复杂度
- 稳定性
