# 项目八：电网建设造价模拟系统

本项目模拟城市小区电网建设：在若干顶点（小区）之间选择 `n-1` 条边，使所有顶点连通且总造价最小，即求图的 **最小生成树（MST）**。

## 核心算法

使用 Prim 算法构造最小生成树：

- 输入：连通无向带权图
- 输出：最小生成树的边集合及总权值（总造价）

实现中使用邻接矩阵存储图。

## 菜单功能

程序为菜单交互方式，常见流程：

1. **A**：创建电网顶点（输入顶点数量与名称）
2. **B**：添加边（输入“顶点1 顶点2 权值”，以 `? ? 0` 结束）
3. **C**：构造最小生成树（输入起始顶点）
4. **D**：显示最小生成树（输出边与总造价）
5. **E**：退出程序

## 输入校验（示例）

- 顶点数量范围检查
- 顶点名称不允许重复
- 边权要求为正整数
- 输入非法时给出提示并可继续
