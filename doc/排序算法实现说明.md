# 排序算法实现说明（项目十）

本文档对项目十中的 8 种排序算法做简要说明，重点关注思想、复杂度与稳定性。

## 复杂度与稳定性对比（简表）

| 算法 | 平均时间复杂度 | 最坏时间复杂度 | 额外空间 | 稳定性 |
| --- | --- | --- | --- | --- |
| 冒泡排序 | O(N²) | O(N²) | O(1) | 稳定 |
| 选择排序 | O(N²) | O(N²) | O(1) | 不稳定 |
| 直接插入排序 | O(N²) | O(N²) | O(1) | 稳定 |
| 希尔排序 | 约 O(N^1.3)~O(N²) | O(N²) | O(1) | 不稳定 |
| 快速排序 | O(N log N) | O(N²) | O(log N)（递归栈） | 不稳定 |
| 堆排序 | O(N log N) | O(N log N) | O(1) | 不稳定 |
| 归并排序 | O(N log N) | O(N log N) | O(N) | 稳定 |
| 基数排序 | O(d·N) | O(d·N) | O(N) | 稳定 |

## 各算法要点

### 冒泡排序

相邻元素比较并交换，较大元素“冒泡”到末尾。可用“本轮是否发生交换”做提前结束优化。

### 选择排序

每一轮在未排序区间中选择最小元素，与当前起始位置交换。

### 直接插入排序

维护已排序区间，将新元素向前插入到合适位置；对“近乎有序”的数据表现较好。

### 希尔排序

按步长 `gap` 分组做插入排序，逐步缩小 `gap`，最后 `gap=1` 结束。

### 快速排序

选取一个枢轴（pivot）分区，小于 pivot 的放左侧，大于的放右侧，递归处理左右区间。

### 堆排序

先建大根堆，然后反复把堆顶（最大值）交换到末尾并调整堆。

### 归并排序

分治：将数组二分递归排序后合并，合并过程线性；稳定但需要额外数组。

### 基数排序

按位（个位、十位、百位……）做稳定的计数分配，属于非比较类排序，对整数排序效果较好。
