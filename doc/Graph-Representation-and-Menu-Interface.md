# 图的表示与菜单交互（项目八）

项目八需要维护一张带权无向图，并通过菜单让用户逐步输入顶点与边。实现选用 **邻接矩阵** 存储图。

## 邻接矩阵

设顶点数为 `V`，用二维数组 `G[V][V]` 表示边权：

- `G[i][i] = 0`
- 若顶点 `i` 与 `j` 有边，则 `G[i][j] = weight`
- 若无边，则 `G[i][j] = INF`（表示不可达）

由于是无向图，矩阵对称：`G[i][j] == G[j][i]`。

## 顶点命名与索引映射

用户输入的是顶点名称（字符串），算法内部需要使用下标：

- 保存一个顶点名称数组 `names[]`
- 通过顺序查找把名称转换为下标（`GetVertexIndex`）

## 菜单交互设计

为便于逐步构建图，采用菜单交互：

- A：创建顶点（清空旧图，重新初始化）
- B：添加边（可重复输入，直到 `? ? 0` 结束）
- C：构造最小生成树（Prim）
- D：显示结果（边与总造价）
- E：退出

## 输入校验建议

- 顶点数范围检查
- 顶点名不重复
- 边两端顶点必须存在
- 边权为正整数
- 不允许自环边（同一顶点到自身）
