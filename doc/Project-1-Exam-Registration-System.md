# 项目一：考试报名系统

本项目模拟考试报名信息管理系统，支持对考生信息进行建立、查找、插入、修改、删除与统计等操作。

## 数据字段

每位考生包含以下信息：

- 准考证号（考号，作为唯一键）
- 姓名
- 性别（男/女）
- 年龄
- 报考类别

## 程序结构（对应 `p1/p1.cpp`）

- `Student`：保存单个考生信息
- `Node`：链表节点（用于处理哈希冲突的拉链）
- `ExamRegistrationSystem`：系统核心类，封装所有功能

系统采用 **哈希表 + 拉链法** 存储考生信息：

- 哈希表桶数为质数（`997`），降低冲突概率
- 同一桶内用单链表保存冲突的记录

## 功能与菜单

程序启动后先进行初始化录入，然后进入菜单循环：

- `1`：插入考生信息
- `2`：删除考生信息
- `3`：查找考生信息（按考号）
- `4`：修改考生信息（考号不可修改）
- `5`：统计/显示全部考生信息
- `0`：退出系统

## 输入校验

为保证交互稳定性，程序对关键输入做了基本合法性检查（示例）：

- 考号：必须为正整数，且在合理范围内
- 性别：仅允许 `男` / `女`
- 年龄：必须为整数，且在合理范围内

## 复杂度（简要）

在哈希分布较均匀的情况下：

- 查找/插入/删除：平均接近 O(1)
- 最坏情况（大量冲突）：退化为链表操作 O(N)

## 编译与运行

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic p1/p1.cpp -o output/p1
./output/p1
```
